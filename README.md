# 🌿 Earth HR - Employee Management System

## Project Overview

**WorkNest** is a responsive Employee Management System developed using **HTML, CSS, and JavaScript**. It provides a simple and intuitive interface to register, view, search, filter, and manage employee records.

The application follows clean coding practices, semantic HTML, responsive web design principles, and dynamic DOM manipulation using JavaScript. Employee data is stored in the browser using **Local Storage**, allowing records to persist even after refreshing the page.

The project features a unique **Earth-inspired theme** along with multiple color themes that users can switch between for a personalized experience.

---

## How the Application Works

### 1. Register Employee

The user fills in the registration form with:

* Employee Name
* Email Address
* Department
* Position
* Salary

After clicking **Register Employee**, the information is:

* validated,
* added to the employee array,
* displayed in the employee table,
* saved to Local Storage,
* and the dashboard statistics are updated.

---

### 2. Employee Table

Every registered employee is displayed dynamically with:

* Employee ID
* Name
* Email
* Department
* Position
* Salary
* Delete Button

---

### 3. Search

Users can search employees by typing the employee name.

The table updates instantly while typing.

---

### 4. Department Filter

Users can filter employee records according to the selected department.

---

### 5. Delete Employee

Clicking the Delete button removes the employee record after confirmation.

The dashboard and Local Storage are updated automatically.

---

### 6. Theme Changer

Users can select different themes from the navigation bar.

The selected theme is remembered using Local Storage and automatically applied the next time the application is opened.
